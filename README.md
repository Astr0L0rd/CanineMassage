# Canine Massage Manager App

This repository contains the source code for **Canine Massage Manager**, a simple React Native application built with Expo. The goal of the project is to help track clients, dogs and massage sessions for small canine massage practices.

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the Expo development server
   ```bash
   npm start
   ```
3. Use the Expo app on your device or an emulator to view the project.

The app stores data locally using AsyncStorage. No external backend is required.

## Features
- Manage clients and their dogs
- Track massage sessions and payments
- Dashboard showing upcoming sessions

This MVP is intentionally simple but ready for expansion (SQLite, cloud sync, calendar export, etc.).
